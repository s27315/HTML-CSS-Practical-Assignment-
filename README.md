# Explore Kigali 🌍

## Project Description
A multi-page website that guides visitors through the best things to do and places to eat around Kigali, Rwanda. The site covers city tours, top restaurants, Nyungwe Forest adventures, and Lake Kivu experiences.

## HTML Pages Included
| Page | Description |
|------|-------------|
| `index.html` | Home page with overview, map, and navigation to all pages |
| `kigali-city-tour.html` | Top attractions and a suggested 1-day city itinerary |
| `restaurants.html` | Best local and international restaurants in Kigali |
| `nyungwe-forest.html` | Trekking, wildlife, and activities in Nyungwe Forest |
| `lake-kivu.html` | Relaxation, water sports, and scenery at Lake Kivu |

## What This Project Demonstrates
- Proper use of semantic HTML5 elements (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Multi-page navigation with internal and external links
- HTML tables, ordered and unordered lists
- Embedded media: video, audio, and Google Maps iframe
- At least 10 images with CSS styling (size, borders, alignment)
- Separate CSS file per page with classes, selectors, and responsive layout
- Clean, readable, and well-indented code structure
